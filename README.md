# ai-regulation-tracker

> Nightly agent monitoring OCC, Federal Reserve, CFPB, FDIC, SEC, FCA, ECB, and EU AI Act feeds for AI-relevant regulatory developments. Published daily to GitHub Discussions.

![Tracker](https://github.com/brianpelow/ai-regulation-tracker/actions/workflows/tracker.yml/badge.svg)

## Follow this repo

**Star and watch this repo** to receive daily AI regulatory intelligence in your GitHub notifications.

Browse past digests in [Discussions](https://github.com/brianpelow/ai-regulation-tracker/discussions) or the [archive/](./archive/) directory.

## What it monitors

| Source | What it tracks |
|--------|---------------|
| OCC | Model risk guidance, AI examination findings, supervisory letters |
| Federal Reserve | SR letters, AI policy statements, model risk management |
| CFPB | Automated decisioning, algorithmic credit, adverse action |
| FDIC | AI in banking, model risk, technology guidance |
| SEC | AI disclosures, algorithmic trading, model governance |
| FCA | AI guidance, model risk, algorithmic accountability |
| ECB | AI in banking supervision, model risk, stress testing |
| EU AI Act | Implementation updates, enforcement, high-risk AI classification |

## What it produces

Each digest contains:

- **New This Week** -- AI-relevant developments from monitored agencies with engineering implications
- **On the Horizon** -- Upcoming comment deadlines, effective dates, proposed rules
- **Engineering Implication** -- One specific, actionable sentence for engineering leaders

## Why this matters

The regulatory landscape for AI in regulated industries is moving faster than most engineering organizations can track. An institution that misses a key guidance update or comment deadline faces examination risk, litigation exposure, and competitive disadvantage.

This tracker ensures engineering leaders in regulated financial services have daily visibility into the regulatory developments that affect how they build and govern AI systems.
